# Ghost Chase Platformer (GitHub Pages)

A tiny pixel-style endless platformer:
- Stickman runner
- Giant ghost chase
- Sword spawns every 20 seconds
- Survive as long as possible

## Controls
- Jump: Space / Arrow Up
- Attack: X (only when sword is active)
- Restart: R

## Run locally
Just open `index.html` in a browser.
